# 导数

## 01 导数

y = f(x)在x0处可导，则：(1)f(x)在x0的领域内有定义；(2)当delta_x趋向0时，delta_y/delta_x的极限存在。

定理：如果f(x)在x0处可导，则f(x)在x0处连续

几何意义：f(x)在x0处的导数等于y=f(x)在(x0, f(x0))处的切线的斜率。切线方程：`y-f(x0)=f'(x0)*(x-x0)`

## 02 常用导数

```
常量的导数为0
(x**a)' = a*x**(a-1)
(a**x)' = (a**x)*ln(a)
(e**x)' = e**x
(sin(x))' = cos(x)
(cos(x))' = -sin(x)
(log(x, a))' = 1/(x*ln(a))
(ln(x))' = 1/x
```

## 03 求导法则

```
(u+v)' = u'+v'
(u*v)' = u'v+uv'
(u/v)' = (u'v-uv')/v**2
(1/v)' = -v'/v**2
y=f(x)及其反函数x=g(y)，则：f'=1/g'
f(g(x))' = f'(x)*g'(x)
```
