# Go发布周期（Go Release Cycle）

原文见：https://github.com/golang/go/wiki/Go-Release-Cycle

Go 现在的发布周期基本是半年一个版本，新的功能特性的开发主要集中在前 3 个月，之后会冻结起来，原则上只接受 bug 修复和文档更新，最终完成测试和发布。
