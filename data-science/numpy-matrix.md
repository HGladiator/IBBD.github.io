# 使用numpy实现矩阵运算

## 矩阵创建

```
In [1]: from numpy import *

In [2]: array([1,2,3])
Out[2]: array([1, 2, 3])

In [4]: mat(a1)
Out[4]: matrix([[1, 2, 3]])

# 创建零矩阵
In [6]: mat(zeros((3,3)))
Out[6]: 
matrix([[0., 0., 0.],
        [0., 0., 0.],
        [0., 0., 0.]])

# 创建1矩阵
In [7]: mat(ones((2,4)))
Out[7]: 
matrix([[1., 1., 1., 1.],
        [1., 1., 1., 1.]])

# 创建随机数矩阵
In [8]: mat(random.rand(2,2))
Out[8]: 
matrix([[0.20391852, 0.63736712],
        [0.27515304, 0.99846676]])

# 创建随机数矩阵
In [9]: mat(random.randint(10,size=(3,3)))
Out[9]: 
matrix([[8, 3, 2],
        [1, 0, 3],
        [9, 4, 8]])

# 创建随机数矩阵
In [10]: mat(random.randint(2,8,size=(2,5)))
Out[10]: 
matrix([[5, 5, 4, 6, 2],
        [5, 5, 6, 3, 6]])

# 创建单位矩阵
In [11]: mat(eye(2,2,dtype=int))
Out[11]: 
matrix([[1, 0],
        [0, 1]])

In [12]: a = [1,2,3]

# 创建对角线矩阵
In [13]: mat(diag(a))
Out[13]: 
matrix([[1, 0, 0],
        [0, 2, 0],
        [0, 0, 3]])
```

## 常见的矩阵运算

```
In [14]: m1 = mat(random.randint(10,size=(3,3)))
In [15]: m2 = mat(random.randint(10,size=(3,3)))

In [16]: m1
Out[16]: 
matrix([[4, 0, 4],
        [8, 3, 9],
        [7, 3, 6]])

In [17]: m2
Out[17]: 
matrix([[4, 2, 3],
        [9, 3, 6],
        [3, 3, 4]])

# 矩阵加法
In [18]: m1 + m2
Out[18]: 
matrix([[ 8,  2,  7],
        [17,  6, 15],
        [10,  6, 10]])

# 矩阵乘法
In [19]: m1 * m2
Out[19]: 
matrix([[28, 20, 28],
        [86, 52, 78],
        [73, 41, 63]])

# 矩阵对应元素相乘
In [31]: multiply(m1, m2)
Out[31]: 
matrix([[16,  0, 12],
        [72,  9, 54],
        [21,  9, 24]])

# 计算逆矩阵
In [21]: m1.I
Out[21]: 
matrix([[ 0.375     , -0.5       ,  0.5       ],
        [-0.625     ,  0.16666667,  0.16666667],
        [-0.125     ,  0.5       , -0.5       ]])

# 矩阵转置
In [22]: m1.T
Out[22]: 
matrix([[4, 8, 7],
        [0, 3, 3],
        [4, 9, 6]])

# 计算矩阵的行列式值
In [23]: linalg.det(m1)
Out[23]: -23.999999999999993
```
