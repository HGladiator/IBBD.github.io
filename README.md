## 使用步骤

发布：

1. git clone本项目
2. 编辑一个文档，markdown格式
3. 生成索引文件`0-index.md`，命令：`./create_index`
4. git add相应的文件
5. 发布博客：`./deploy '说明'`

搜索：

1. 按关键词搜索：`./search your-keyword`
